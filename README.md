# ElPro Scripts - Premium Roblox Scripts Hub

## Security & Production Ready

This is a production-grade static website with:
- **SEO Optimized**: Meta tags, Open Graph, structured data
- **Security Hardened**: CSP, secure headers, XSS protection
- **Performance**: Lazy loading, optimized assets
- **Accessibility**: Semantic HTML, ARIA labels
- **PWA Ready**: Web manifest for app installation

## Features

- 6+ Premium Roblox Scripts
- Master Sensei Hub (Universal Key System)
- Responsive Design
- Dark Mode Optimized
- Particle Background Animations
- Modal Script Preview System

## Quick Start

```bash
npm install
npm run dev    # Local development
npm run start  # Production serve
```
